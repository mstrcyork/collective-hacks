Music, Science and Technology Research Centre at YorNight 2020

This interface looks for barcodes in a poster (A4 version; A0 original) and uses recognised codes to trigger samples (pre-recorded sounds) or synthesiser tones.

If you'd like to learn more about music and coding for the web, visit this page.

(This interface is a proof-of-concept hack. Apologies for any bugs, lack of style,...)
