Music, Science and Technology Research Centre at YorNight 2020

This interface can be used to take a photo and turn darker patches into sound. Sounds vary in frequency (pitch) according to vertical position in the photo and over time according to horizontal position. This interface is an example of sonification – the general process of turning data into sound.

If you'd like to learn more about music and coding for the web, visit this page.

(This interface is a proof-of-concept hack. Apologies for any bugs, lack of style,...)
